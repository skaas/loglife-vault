# Telegram

이 폴더는 텔레그램에서 들어온 기본 raw 입력을 날짜별로 저장한다.

## 포함되는 것

- 텍스트 메모
- 짧은 생각과 회고
- 사진 공유 링크
- 웹 링크
- 생활 로그

사진과 웹 기록은 별도 폴더로 분리하지 않고, 텔레그램 기록 안에서 함께 다룬다.
이미 Google Photos와 연결된 링크도 이 흐름 안에서 추적한다.

## 원칙

- 파일 하나는 메시지 하나를 기준으로 둔다.
- 메타데이터는 가능한 한 유지한다.
- 링크가 포함된 메시지는 원문 URL을 남기고, 링크 요약은 raw 바깥의 파생 메모로 만든다.
- 이 폴더는 검색 가능한 원본 저장소이며, 의미 연결과 모델 업데이트는 `Daily`와 `Wiki`에서 만든다.
