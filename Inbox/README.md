# Inbox

이 폴더는 외부 입력이 처음 도착하는 원본 보관 구역이다.

## 목적

- 가공 전 기록을 잃지 않고 남긴다.
- 나중에 다시 해석하거나 재색인할 수 있도록 입력 흔적을 보존한다.
- 링크가 포함된 입력은 나중에 내용을 확인하고 요약할 수 있게 한다.

## 원칙

- 원본은 가능하면 수정하지 않는다.
- 정리와 연결은 `Daily`와 `Wiki`에서 한다.
- 현재 입력 채널은 `Telegram`과 `Text`다.
- 텍스트 안에 링크가 있으면 링크를 열어 핵심 내용을 짧게 요약한다.
- 링크 요약은 원본과 분리된 해석 메모로 남긴다.
