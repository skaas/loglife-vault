# Inbox

이 폴더는 외부 입력이 처음 도착하는 관찰 로그 계층이다.

## 목적

- 가공 전 기록을 잃지 않고 남긴다.
- 나중에 다시 해석하거나 재색인할 수 있도록 입력 흔적을 보존한다.
- 링크가 포함된 입력은 나중에 compile에서 요약할 수 있게 원문 상태로 보존한다.

## 원칙

- 원본은 가능하면 수정하지 않는다.
- 이 폴더의 역할은 보관이지 의미 확정이 아니다.
- 정리와 연결은 `Daily`와 `Wiki`에서 한다.
- 현재 입력 채널은 `Telegram`과 `Text`다.
- 링크 요약은 raw 원본을 덮어쓰지 않고, compile에서 만들어지는 분리된 해석 메모로 남긴다.
