# Text

이 폴더는 텔레그램이 아닌 일반 텍스트 입력을 저장한다.

## 포함되는 것

- 직접 적은 짧은 메모
- 복사해 둔 문장
- 기사나 문서 링크와 함께 남긴 생각
- Google Docs 같은 외부 문서에서 가져온 이력서, 프로필, 자기소개 스냅샷
- 임시 기록

## 링크 처리

- 텍스트 안에 링크가 있으면 원문 URL을 그대로 보존한다.
- 가능하면 링크를 열어 내용을 확인하고 핵심을 짧게 요약한다.
- 요약에는 무엇에 대한 글인지, 핵심 주장이나 정보가 무엇인지, 왜 이 기록과 연결되는지를 적는다.
- 긴 요약보다 짧고 검색 가능한 메모를 우선한다.

## 원칙

- 원본 텍스트는 가능한 한 수정하지 않는다.
- 링크 요약은 원본을 덮어쓰지 않고, 보조 메모로 남긴다.
- 의미 연결과 재분류는 `Daily`와 `Wiki`에서 한다.
