# Questions

이 폴더는 열린 질문, 가설, 이어서 생각해야 할 실마리를 둔다.

## 예시

- 왜 특정 상황에서 반복적으로 같은 반응을 보일까
- 요즘 내가 중요하게 여기는 것은 무엇일까
- 어떤 관계가 나를 편하게 만들거나 지치게 하는가
- 방금 공부한 개념을 내 일과 삶에 어떻게 연결할 수 있을까

## 목적

- 결론보다 탐색 과정 자체를 보존한다.
- 이후 기록이 들어왔을 때 어떤 질문과 연결되는지 다시 걸 수 있게 한다.
- 개념 정리와 추가 생각거리를 한 파일 안에서 같이 다룰 수 있다.
- 닫히지 않은 질문은 실패가 아니라 모델이 아직 열려 있다는 신호로 본다.
