# 미분의-변수-해석

이 문서는 2026-04-09에 붙잡은 개념 메모를 다시 정리한 것이다.

## 핵심 정리

- 미분은 시간 전용 개념이 아니다.
- `x`는 시간뿐 아니라 가격, 난이도, UI 복잡도 같은 어떤 변수도 될 수 있다.
- 핵심 의미는 같다. `x`를 조금 바꿨을 때 `y`가 얼마나 반응하느냐를 보는 것이다.
- 그래서 시간축에서 보면 `트렌드`처럼 읽히고, 시간 말고 다른 축에서 보면 `민감도`, `기울기`, `영향력`처럼 읽힌다.

## 예시로 다시 말하면

- 가격과 수요:
  가격을 조금 올렸을 때 판매량이 얼마나 줄어드는가
- 게임 난이도와 재미:
  난이도를 올렸을 때 재미가 더 커지는가, 떨어지는가
- UI 복잡도와 전환율:
  버튼 하나, 단계 하나를 늘렸을 때 전환율이 어떻게 움직이는가

## 내 기록과 연결

- 이 개념은 최근 메모에 반복해서 나온 `브랜드`, `해저드커브`, `RFM` 같은 분석 언어와 잘 이어진다.
- 결국 공통 질문은 하나다. `내가 어떤 변수를 바꿨을 때, 어떤 결과가 얼마나 반응하느냐`다.
- 그래서 이 메모는 수학 공부 자체보다 제품 기획과 밸런스 감각을 다시 번역하는 메모로 보는 편이 맞다.

## 더 생각할 것

- 내가 지금 실제로 조절할 수 있는 `x`는 무엇인가.
- 내가 정말 보고 싶은 `y`는 재미, 전환율, 유지율, 매출 중 무엇인가.
- 지금 보는 관계는 선형인가, 아니면 임계점이나 최적점이 있는가.
- `2차 미분`처럼 변화의 변화까지 봐야 하는 문제는 무엇인가.
