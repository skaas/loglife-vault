# Self

이 폴더는 당신 자신에 관한 누적 지식을 보관한다.

## 목적

- 자기서술을 흩어진 메모가 아니라 하나의 구조로 모은다.
- 가치, 욕구, 성격, 한계, 질문, 긴장을 장기적으로 추적한다.
- 다른 `Themes`, `People`, `Projects` 문서와 연결되는 중심 허브가 된다.

## 기본 문서

- `Map.md`:
  자기지식의 전체 지도
- `Profile.md`:
  비교적 안정적인 자기서술
- `Tensions.md`:
  반복되는 내적 충돌
- `Open Questions.md`:
  아직 닫히지 않은 질문
