# Self

이 폴더는 당신 자신에 관한 누적 지식을 보관한다.

## 목적

- 자기서술을 흩어진 메모가 아니라 하나의 구조로 모은다.
- 가치, 욕구, 성격, 한계, 질문, 긴장을 장기적으로 추적한다.
- 다른 `Themes`, `People`, `Projects` 문서와 연결되는 중심 허브가 된다.

## 기본 문서

- `Map.md`:
  자기지식의 전체 지도
- `Profile.md`:
  비교적 안정적인 자기서술
- `Current Diagnosis.md`:
  최근 기록 기준 현재 상태 진단
- `Today.md`:
  오늘 하나만 잡는 실행 포커스 또는 짧은 글쓰기 프롬프트
- `TODO.md`:
  최근 기록과 진단에서 나온 다음 행동
- `Career.md`:
  이력서와 경력 문서에서 추출한 직업 정체성
- `Speaking.md`:
  강연과 발표 자료에서 추출한 설명 주제와 방식
- `Tensions.md`:
  반복되는 내적 충돌
- `Open Questions.md`:
  아직 닫히지 않은 질문
