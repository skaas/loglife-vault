# Wiki

이 폴더는 원본 기록에서 추출한 장기 자기 작업 모델이다.

## 시작점

- [index.md](<index.md>):
  현재까지 컴파일된 자기지식의 요약 목차
- [Self/Map.md](<Self/Map.md>):
  자기지식 중심 지도
- [Ideas/README.md](<Ideas/README.md>):
  창작, 세계관, 게임 아이디어를 모으는 축

## 목적

- 사람, 사건, 주제, 질문, 프로젝트 같은 축으로 기록을 다시 묶는다.
- 창작과 서사 아이디어도 장기적으로 다시 찾을 수 있는 노드로 정리한다.
- `Self`를 중심 허브로 두고, 나에 관한 지식이 누적되게 만든다.
- 단순 검색이 아니라, 생각과 생각 사이의 연결을 남긴다.
- 시간이 지나도 이어지는 패턴을 추적한다.
- 완성된 설명서가 아니라, 사용자와 에이전트가 함께 갱신하는 살아 있는 모델로 다룬다.

## 작성 원칙

- 가능한 한 원본 근거를 남긴다.
- 사실과 해석을 섞지 않는다.
- 짧더라도 링크를 통해 다른 문서와 연결한다.
- 새 기록은 기존 노드를 갱신하는 방향으로 우선 반영한다.
- 모순은 삭제보다 `Tensions`, `Open Questions`, queue로 드러내는 쪽을 우선한다.
