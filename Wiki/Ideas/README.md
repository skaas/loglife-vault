# Ideas

이 폴더는 창작, 세계관, 게임, 서사 아이디어를 짧은 노드로 누적한다.

## 목적

- 아직 프로젝트로 확정되지 않은 아이디어를 버리지 않고 다시 찾을 수 있게 한다.
- `Projects`처럼 일정과 액션을 추적하기보다, 핵심 설정과 질문을 짧게 보존한다.
- 비슷한 아이디어의 중복 초안을 한 노드로 모은다.

## 다루는 내용

- 게임 시놉시스
- 세계관 메모
- 서사 장치
- 철학적 대비 구도
- 확장할 가치가 있는 짧은 아이디어

## 원칙

- 원본 표현을 완전히 다듬기보다 핵심 아이디어를 짧게 압축한다.
- 같은 아이디어의 중복 초안은 한 페이지로 모은다.
- 아직 실행 단계가 아니면 `Projects`로 올리지 않는다.
