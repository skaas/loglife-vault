# Site

이 폴더는 내부 작업 모델의 일부를 외부용으로 투영한 산출물을 둔다.

## 목적

- 내부 위키를 그대로 공개하지 않고, 외부에 보여도 되는 내용만 모은다.
- 이력서, 포트폴리오, 소개 페이지처럼 `나란 사람`을 짧게 설명하는 결과물을 만든다.
- 시스템의 중심은 아니며, `Wiki`에서 안전하게 추린 결과만 반영한다.

## 기본 규칙

- 원본 `Inbox`와 `Daily`는 직접 사용하지 않는다.
- `Wiki`에서 공개 가능한 문서만 골라 다시 요약한다.
- 가족, 건강, 아이 관련 세부 내용, 내적 충돌, 미해결 질문은 제외한다.
- 날짜나 직함에 버전 차이가 있으면 확정적으로 쓰지 말고 완화하거나 생략한다.
- 직접 연락처는 기본적으로 이메일만 사용하고, 전화번호는 공개 기본값에서 뺀다.
- 공개 문장은 `Meta/wording-map.yaml`의 정규 표현을 우선 사용한다.
- 같은 의미를 다른 워딩으로 쓸 경우에는 `Meta/wording-conflicts.md`에 open conflict를 남긴다.

## 현재 산출물

- `index.html`
  공개용 프로필 겸 자기소개 페이지
- `tasks.html`
  `Wiki/Self/TODO.md`와 `Wiki/Self/Today.md` 기반 실행 보드. 완료 기록과 짧은 생각 기록을 `Inbox/Text`에 생성
- `styles.css`
  공개 페이지 전용 스타일
- `tasks.css`
  실행 보드 전용 스타일
- `tasks.js`
  Vault 연결, Today/TODO 파싱, 짧은 생각 저장, 완료 기록 저장 로직
