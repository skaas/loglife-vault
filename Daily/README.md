# Daily

이 폴더는 날짜별 문맥 압축본을 저장하는 중간 작업 메모리다.

## 목적

- 하루 동안 들어온 기록을 시간순으로 훑어볼 수 있게 한다.
- 원본 메시지를 그대로 복사하기보다, 그날의 흐름과 맥락을 정리한다.
- 필요한 경우 `Wiki` 문서로 연결되는 허브 역할을 한다.
- 아직 장기 모델로 승격하기 이른 해석과 후보를 임시로 담을 수 있다.

## 권장 방식

- 파일은 날짜 기준으로 관리한다.
- 짧은 사실 기록과 함께, 그날 떠오른 해석이나 질문을 덧붙일 수 있다.
- 중요한 항목은 관련 `Wiki` 문서 링크를 남긴다.
- `Daily`는 장기 모델의 본체가 아니라, 승격 전 문맥 계층으로 본다.
